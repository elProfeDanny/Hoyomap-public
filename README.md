# HoyoMap – Chile sin baches

Plataforma con IA que predice deterioro vial usando reportes ciudadanos con foto + GPS.

### Estado actual (Noviembre 2025)
- Web 100 % funcional: hoyomap.cl  
- 5 reportes reales en Maipú (TRL4 validado)  
- Mapa de calor en tiempo real con Leaflet + Leaflet.heat  
- Backend PHP/MySQL + API pública  
- Postulando a Start-Up Chile Build BIG11
